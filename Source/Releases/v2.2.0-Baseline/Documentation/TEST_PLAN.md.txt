# Regression Checklist

Before every release:

* Project compiles successfully.
* No broken references.
* Database opens.
* Sample Resource Email processed.
* Sample Offer Email processed.
* Sample Interview Email processed.
* Sample BGV Email processed.
* Sample NATS Email processed.
* Sample NAPS Email processed.
* Candidate Timeline updated.
* No duplicate records created.

---

# Release Approval Checklist

Before marking a release complete:

* All unit tests passed.
* Integration tests passed.
* Compile successful.
* Database backup created.
* VBA modules exported.
* CHANGELOG updated.
* RELEASE_NOTES updated.
